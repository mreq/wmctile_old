wmctile

Window manager's best friend in a gem.

About wmctile

Wmctile was created because of the regular set of Desktop Environments lacking crucial features such as:

  • Switching windows based on application names.
  • Snapping windows to various sizes anywhere on the screen.
  • Resizing windows by a certain amount.
  • Tiling windows to fill the screen.

Wmctile enables you to do that and much more.

What's with the name?

It's a mix of the words wmctrl and tile, former being the key utility wmctile uses and latter a word related to tiling window managers.

Getting started

Easy distribution and installation is the main reason why wmctile is written in Ruby. It's bundled as a gem, so the actual installation couldn't be easier.

Installation

  1. Install required packages (wmctrl, xrandr, dmenu). Ubuntu has all these in the standard repositories so it's as easy as writing:

    sudo apt-get install wmctrl suckless-tools

    You'll need Ruby and RubyGems installed as well. If you need help, check the rubygems guide.

  2. Install the wmctile gem:

    gem install wmctile

  3. That's it :) Check the help command or the docs to see what you can do.

    wmctile help
