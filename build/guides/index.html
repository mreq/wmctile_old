Wmctile guides

This page contains simple examples of using wmctile.

General information

Wmctile uses wmctrl to switch, move or arrange windows. It's a command-line utility, not a daemon.

Basic command and arguments

The basic form of a wmctile command is:

wmctile [--option1, --option2, ...] <command> ['argument1', 'argument2', ...]

Available commands are:

  • summon and summon_or_run
  • switch_to and switch_to_or_run
  • maximize and unmaximize
  • shade and unshade
  • snap
  • resize

An example command, snapping a Thunderbird window to the left half of the screen would be:

wmctile snap 'left' 'Mail.Thunderbird'

One of the arguments is usually a window_string which is given in the form wmctrl uses. To see what wmctile uses, let's look at what it outputs:

wmctrl -lx
0x00600004 -1 xfce4-panel.Xfce4-panel  hostname xfce4-panel
0x01000003 -1 xfdesktop.Xfdesktop   hostname Desktop
0x06000003  1 sublime_text.Sublime_text  hostname ~/whatever/path
0x0620001a  2 scudcloud.Scudcloud   hostname ScudCloud
0x066000a6  2 Mail.Thunderbird      hostname Inbox - host@name.com - Mozilla Thunderbird

When looking up applications, wmctile simply greps the output shown above. The relevant part is the middle one, as it's unique to each application, allowing the user to identify different applications (the title can't be used as it may change at any time).

Using wmctile

Switching windows

Commands used:

  • summon and summon_or_run
  • switch_to and switch_to_or_run

Wmctile distinguishes between switching to and summoning an application. Switching to an application means going to the application's workspace and raising the window. Summoning an application brings it to your current workspace and then raises it. There's no difference if you're working on a single workspace or if you don't use the --all-workspaces option.

Switching to a window

wmctile switch_to 'blue.Window'
Switching to a window/running a command
When a window exists:
wmctile switch_to_or_run 'blue.Window' 'launch_blue_window_cmd'
When it does not:
wmctile switch_to_or_run 'blue.Window' 'launch_blue_window_cmd'

Summoning a window

wmctile summon 'blue.Window'
Summoning a window/running a command
When a window exists:
wmctile summon_or_run 'blue.Window' 'launch_blue_window_cmd'
When it does not:
wmctile summon_or_run 'blue.Window' 'launch_blue_window_cmd'

Moving windows around

Snapping window

snap where 'window_string' ['portion']

You can snap window top, right, bottom or left. You may pass an additional portion argument to tell the script how wide/tall the window should be. Defaults to 0.5.

wmctile snap left 'red.Window'
wmctile snap right 'red.Window' 0.3
wmctile snap top 'red.Window' 0.7
wmctile snap bottom 'red.Window' 0.4
