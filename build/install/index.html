Installation

  1. Install required packages (wmctrl, xrandr, dmenu). Ubuntu has all these in the standard repositories so it's as easy as writing:

    sudo apt-get install wmctrl suckless-tools

    You'll need Ruby and RubyGems installed as well. If you need help, check the rubygems guide.

  2. Install the wmctile gem:

    gem install wmctile

  3. That's it :) Check the help command or the docs to see what you can do.

    wmctile help
